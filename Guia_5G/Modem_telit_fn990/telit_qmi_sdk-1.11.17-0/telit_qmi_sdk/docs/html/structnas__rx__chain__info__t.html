◆ is_radio_tuned

bool nas_rx_chain_info_t::is_radio_tuned

Whether Rx is tuned to a channel: • 0x00 – Not tuned • 0x01 – Tuned

If the radio is tuned, instantaneous values are set for the signal information fields below. If the radio is not tuned, or is delayed or invalid, the values are set depending on each technology.
