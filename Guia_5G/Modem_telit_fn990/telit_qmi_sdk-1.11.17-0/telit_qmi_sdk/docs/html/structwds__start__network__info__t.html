◆ authentication_pref

uint8_t wds_start_network_info_t::authentication_pref

bitmap indicating the algorithm preference, according to the following values: Bit 0 - PAP preference:

  • 0 - PAP is never performed
  • 1 - PAP may be performed Bit 1 - CHAP preference:
  • 0 - CHAP is never performed
  • 1 - CHAP may be performed If more than one bit is set, the device decides which authentication procedure is performed while setting up the data session.
