bool loc_delete_gnss_service_data_t::delete_all

Indicates whether to delete all GNSS service data.

Values: OxO1 (TRUE): Reset all constellations’ service data; if this flag is set, all the other information contained in the optional fields for this messag are ignored. 0x00 (FALSE): The optional fields in the message are used to determine which data to delete
