◆ report_period

uint32_t nas_set_nr5g_sync_pulse_gen_t::report_period

Report Period. Configures pulse generation indication periodicity. Range: 0 - 128. The value is a multiple of 10 milliseconds. For example, if the value is set to 1, the report is sent every 10 milliseconds. If the value is set to 128, the indication is sent every 1280 milliseconds. If this TLV is not set or the value is set to 0, this indication is disabled
