◆ lpm_flag

uint32_t gms_dms_get_pc_info_resp_t::lpm_flag

LPM force flags

Values (Bitmask): bit0 - Tracks the state of W_DISABLE TLV, indicating state of the W_DISABLE switch bit1 - Set if LPM is requested by host request, such as AT command, QMI or MBIM request. Cleared by a host request to return to online mode. Also set when the device is cold or warm booted in persistent LPM. bit2 - Set when the device temperature is outside the valid operating range. Cleared if the temperature returns to the normal range. bit3 - Set when the device voltage is outside the valid operating range. Cleared if the voltage returns to the normal range. bit4 - Set on power up when BIOS locking is enabled. Cleared when the host has disabled the BIOS lock. (Not Supported) bit 5- Set if the current device configuration does not match the GOBI image preference. (Not Supported) bit6-31 - Additional LPM causes may be added to future products. (Not supported)
