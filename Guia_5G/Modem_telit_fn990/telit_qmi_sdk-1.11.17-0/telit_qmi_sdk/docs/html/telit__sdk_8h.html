◆ strlcpy()

size_t strlcpy ( char *  dst,
const char *  src,
size_t  size 
)

Size-bounded string copying.

The strlcpy() function copy strings. It is designed to be safer, more consistent, and less error prone replacement for strncpy. Unlike that function, strlcpy() take the full size of the buffer (not just the length) and guarantee to NUL-terminate the result (as long as size is larger than 0). Note that a byte for the NUL should be included in size. Also note that strlcpy() only operate on true “C” strings. This means that for strlcpy() src must be NUL-terminated.

The strlcpy() function copies up to size - 1 characters from the NUL-terminated string src to dst, NUL-terminating the result.

Parameters
dsta buffer where the string is copied
srca string to be copied
sizethe size of the destination buffer
Returns
length of the string to be copied
