◆ delete_all_flag

bool loc_delete_assist_data_t::delete_all_flag

Indicates whether all assistance data is to be deleted.

Valid values:

  • 0x01 (TRUE) – All assistance data is to be deleted; if this flag is set, all the other information contained in the optional fields for this message are ignored
  • 0x00 (FALSE) – The optional fields in the message are to be used to determine which data is to be deleted
