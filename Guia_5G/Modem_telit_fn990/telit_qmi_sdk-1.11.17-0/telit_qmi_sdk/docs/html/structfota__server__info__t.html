char fota_server_info_t::delta_descriptor_path[QMI_FOTA_MAX_URL_STRING_SIZE]

Root path where fota delta descriptor is gathered If this value is not defined, ftp home directory will be used. For example, if delta descriptor is uploaded to the ftp server in the following path FOTA_DELTA/<MODEL>/<CURRENT_SW_VER>/delta_descriptor.txt this field shall be filled with the value "FOTA_DELTA"
